Root Reign : Global Game Jam 2023

Root Reign is a Snake-style game mixed with Point Capture. One-on-one, the aim is to grab as many objects as possible while rooting your territory. The time limit is one minute. The game can be played in local multiplayer mode, using a joystick and only on PC. Movement is automatic, so use your joystick to change direction. Root Reign is the ideal game for a fun evening with friends on the couch.

In this project, I developed mechanics for the main character

Paper Feel : End of the first year project

Project realized in group at the end of the first year of GameDesign, the theme was "FeelGood" and "cinematic platformer". Our objective was therefore to produce a game in two months

In this project, I developed mechanics for the main character, as well as assets integration and Level Building

I was able to acquire new knowledge with this project, first of all group work, organization and team leadership. I was also able to improve my coding skills, and the use of unity tools such as scene lighting or the animator

PROJECT : Magnet Person

PROJECT: Magnet Person is a First Person Puzzle Game in a Futuristic 3D Universe. We embody the subject 710N37, victim of an experiment that transformed him into a magnetic man, he can modify his polarity to interact with the environment. His goal will be to escape from this lab using his newly acquired powers, without pressure.

This project was designed and produced for the Global GameJam 2022 on the theme Duality.

After the jam I took over the project for a class project, so I reworked the game mechanics and focused on the level design and gamefeel.
